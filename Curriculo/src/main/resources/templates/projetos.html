Projeto de IA (Identificacao de cogumelo venenoso e comestivel)

Projeto de Machine Learning em Python, onde utilizei bibliotecas como Pandas, Sklearn e Numpy para desenvolver um modelo capaz de identificar se um cogumelo é venenoso ou comestível com base em seu odor. O modelo foi treinado e testado com um conjunto de dados público, e mostra que classificar apenas pelo odor não e o suficiente para termos uma super precissão. Este projeto me permitiu aprofundar meus conhecimentos em ciência de dados e aprendizado de máquina, além de aprimorar minhas habilidades em programação Python.

Tecnologias utilizadas: Python, Pandas,SVM,Matriz de confusao, Sklearn, Numpy.

Link:www.github.com/bielxcesar/SVM_Cogumelo_odor
